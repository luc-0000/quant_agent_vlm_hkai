## Data Sources

| Stage | Source | Method |
|-------|--------|--------|
| 股票列表 | HK AI MCP | `list_selectable_stocks()` |
| 实时行情 | HK AI MCP | `get_quote_by_symbols()` |
| K 线数据 | HK AI MCP | `get_stock_kline()` |
| 技术指标 | 本地 TA-Lib | MCP tools (compute_rsi / macd / stoch / roc / willr) |
| K 线图表 | 本地 mplfinance | MCP tools (generate_kline_image / generate_trend_image) |

## LLM Usage

| Stage | Model | Purpose |
|-------|-------|---------|
| 选股 | `qwen-plus-latest` (DashScope) | 从候选列表挑最有潜力的 1 只 |
| 指标汇总 | `qwen-plus-latest` | 解读 RSI/MACD/Stoch/ROC/WillR |
| 形态识别 | `qwen3-vl-plus` (VLM) | 看 K 线图识别头肩顶/双底等 16 种形态 |
| 趋势分析 | `qwen3-vl-plus` (VLM) | 看趋势线图分析支撑/阻力 |
| 最终决策 | `qwen-plus-latest` | 综合三个报告输出 LONG/SHORT/HOLD |
